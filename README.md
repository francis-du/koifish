# Contributing

## Code 

1. install rust 

2. install cargo

3. run

```shell script
cargo run 
```

## Docs

1.  install mdbook

```shell script
cargo install mdbook 
```
 
2.  write Markdown docs under the `docs/src`

3.  run

```shell script
mkbook serve
```
